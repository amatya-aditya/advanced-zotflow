# ZotFlow Documentation

Welcome to the ZotFlow documentation. These guides will help you get up and running quickly.

## Overview

ZotFlow connects **Zotero** (your reference manager) with **Obsidian** (your note-taking app). Here's how they work together:

- **Zotero** holds your library — papers, books, metadata, and annotations.
- **ZotFlow** syncs that library into Obsidian and lets you read and annotate attachments without leaving your vault.
- For each Zotero item, ZotFlow generates a **Source Note** — a Markdown file containing the item's metadata and your annotations.

### Source Notes Are Read-Only

Source notes are **auto-generated and locked**. You should not edit them directly — any manual changes will be overwritten the next time ZotFlow updates the note (e.g., after a sync or a new annotation).

Instead, create your own separate notes and **link back** to the source note. This keeps a clean boundary: source notes capture _what the author said_, your notes capture _what you think_.
